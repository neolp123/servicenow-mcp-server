# ServiceNow MCP Server

Model Context Protocol server for ServiceNow with SSE transport.

## Deployment

Deploy to Render with one click!

## Environment Variables

- `SERVICENOW_INSTANCE_URL` - Your ServiceNow instance
- `SERVICENOW_USERNAME` - API username
- `SERVICENOW_PASSWORD` - API password
- `MCP_TOOL_PACKAGE` - Tool package (default: full)
